\subsection{Detailed Description}
\doxyref{CM}{p.}{classCM} Class. 

This class provides four coded modulation schemes:\begin{itemize}
\item \doxyref{TCM}{p.}{cm_8h_33275405824294f33926d36091cc4d27}, \doxyref{TTCM}{p.}{cm_8h_ad4bfaba09cc6a6f69d2a48fa0423110}, \doxyref{BICM}{p.}{cm_8h_1218f70f86ea4acd32d0b58267a37b30} and \doxyref{BICMID}{p.}{cm_8h_68291ce81955ca351b54cc17eed37a7b}.\end{itemize}


Modulation modes available are M-QAM and M-PSK, where {\em M=2$^\wedge$n\/} is the modulation level. We have:\begin{itemize}
\item 4-QAM, 16-QAM, 64-QAM (\doxyref{CQAM}{p.}{cm_8h_53b2e3c18e665091f2d8a3aa4bd9e68c})\item 4-PSK, 8-PSK, 16-PSK, 64-PSK (\doxyref{CPSK}{p.}{cm_8h_07f2a000bb7ecef9011cdc5a02658cce})\end{itemize}


Interleaver types available are:\begin{itemize}
\item symbol {\em interleaver\/} (\doxyref{ON}{p.}{cm_8h_d76d1750a6cdeebd506bfcd6752554d2}, \doxyref{OFF}{p.}{cm_8h_29e413f6725b2ba32d165ffaa35b01e5}) for TCM, TTCM\item internal parallel bit interleavers for \doxyref{BICM}{p.}{cm_8h_1218f70f86ea4acd32d0b58267a37b30} and \doxyref{BICMID}{p.}{cm_8h_68291ce81955ca351b54cc17eed37a7b} (always \doxyref{ON}{p.}{cm_8h_d76d1750a6cdeebd506bfcd6752554d2} for \doxyref{BICM}{p.}{cm_8h_1218f70f86ea4acd32d0b58267a37b30} \& \doxyref{BICMID}{p.}{cm_8h_68291ce81955ca351b54cc17eed37a7b})\item IQ {\em interleaver\/} (\doxyref{IQ\_\-INTLV}{p.}{cm_8h_0e8351180111f8b1e0bf7c7484154684}) for TCM, TTCM, BICM, BICMID\end{itemize}


Decoding in logarithmic domain is performed: {\em decoder\_\-type\/} available are \doxyref{CM\_\-EXACT\_\-LOG\_\-MAP}{p.}{cm_8h_153b4193dd1c76575af0d2e1f56c9ba5}, \doxyref{CM\_\-APPROX\_\-LOG\_\-MAP}{p.}{cm_8h_7eb345f72a74fb180af11b6a150488cf}, \doxyref{CM\_\-MAX\_\-LOG\_\-MAP}{p.}{cm_8h_f071e75e173cacdf76de0b0e09a0e635}.

To set up a rate-3/4 ({\em k=3\/}, {\em n=4\/}) 16QAM TTCM having code memory 3, code termination, 4 turbo iteration, \doxyref{CM\_\-APPROX\_\-LOG\_\-MAP}{p.}{cm_8h_7eb345f72a74fb180af11b6a150488cf} decoder, 100 coded symbols ({\em no\_\-of\_\-symbols\/}), employ IQ-channel interleaver; the following code can be used: 

\begin{Code}\begin{verbatim}  CM cm;
  cm.set_parameters(TTCM,3,4,3,ON,4,CM_APPROX_LOG_MAP,100,CQAM,IQ_INTLV);
  cm.initialise();
\end{verbatim}
\end{Code}

 See \doxyref{cm\_\-test.cpp}{p.}{cm__test_8cpp} for more details.
